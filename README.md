# Blind Teknologis

Enterprise marketing website for [blindteknologis.com](https://blindteknologis.com).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```

GitHub Pages via GitHub Actions on push to `main`.
